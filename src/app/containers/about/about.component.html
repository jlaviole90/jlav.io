About this Site

This began as a project to learn Angular animations; I've been working with Angular for years, but enterprise CRUD apps rarely use animations. I figure keeping it (in all its flaws) and using it as a resume and portfolio site is better than not having one at all.

  • The landing page uses a mix of both CSS keyframes and Angular animations.
  • The dashboard page is actually 'animated' entirely with observables.
  • This page is as barebones as possible. Oldschool, reminiscent of a better internet.
